



## 扩展

### 美化与主题类

- **Blur my Shell**：给 GNOME Shell 界面（顶栏、概览界面等）添加毛玻璃模糊效果，提升视觉质感。
- **Just Perfection**：高度自定义 GNOME Shell 外观与行为，可隐藏面板元素、调整布局、优化交互，是 “美化党” 的核心扩展。
- **Light Style** / **Night Theme Switcher**：分别用于浅色主题和自动切换日夜主题，后者可根据时间或日出日落自动切换明暗模式。
- **User Themes**：允许加载自定义 GNOME Shell 主题，突破系统默认主题限制。
- **Status Icons**：在顶栏显示系统状态图标（如网络、音量等），可自定义显示样式。


### 窗口与工作区

- **Auto Move Windows**：自动将新打开的窗口移动到预设的工作区或屏幕位置，适合多工作流场景。
- **Native Window Placement**：优化窗口初始打开位置，避免默认居中或随机位置，让窗口布局更符合直觉。
- **Window List** / **Window Navigator**：Window List 是传统窗口列表（类似任务栏），Window Navigator 提供更高效的窗口切换与预览。
- **Workspace Indicator**：在顶栏显示当前工作区编号，方便多工作区切换时定位。
- **Dash to Dock**：将 GNOME 内置的 Dash（应用启动器）变成常驻底部 Dock，类似 macOS 的程序坞，提升应用启动效率。


### 系统与实用工具

- **AppIndicator and KStatusNotifierItem Support**：兼容传统应用指示器（如微信、QQ、Steam 等托盘图标），解决部分应用托盘不显示的问题
- **Apps Menu**：在顶栏添加一个应用菜单按钮，可快速浏览、分类查找已安装应用。
- **Caffeine**：临时禁用屏幕自动休眠和锁屏，适合看视频、演示等场景。
- **Clipboard History**：记录剪贴板历史，支持快速粘贴之前复制的内容，提升复制粘贴效率。
- **Launch New Instance**：点击 Dock 或应用图标时，总是打开新窗口，而不是切换到已有窗口。
- **Places Status Indicator**：在顶栏添加快速访问书签、挂载盘、常用文件夹的菜单。
- **Removable Drive Menu**：显示可移动存储设备（U 盘、移动硬盘），支持快速挂载 / 弹出。
- **Screenshot Window Sizer**：截图时可精确调整窗口大小，方便制作规范尺寸的截图。
- **System Monitor** / **Vitals**：两者都是系统监控工具，Vitals 功能更全面，可在顶栏实时显示 CPU、内存、网速、温度等硬件信息；System Monitor 侧重系统资源可视化。

## 常用命令

- 查看所有内置主题：`ghostty +list-themes`
- 查看默认配置：`ghostty +show-config --default`
- 列出可用字体：`ghostty +list-fonts`

## 快捷键

| 快捷键 | 功能 |
| --- | --- |
| `Super` + `W` | 关闭当前标签页 |
| `Super` + `T` | 新建标签页 |
| `Super` + `D` | 向右分屏 |
| `Super` + `Shitf` + `D` | 向下分屏 |
| `Alt` + `↑` / `↓` / `←` / `→` | 切换到上下左右分屏 |
| `Super` + `Shift` + `e` | 分屏平均分配大小 |
| `Super` + `Shift` + `enter` | 当前分屏全屏 / 还原 |
| `Ctrl` + `+` / `-` / `0` | 增大 / 减小 / 重置字号 |
| `Super` + `Shift` + `,` | 重载配置 |

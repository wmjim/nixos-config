# Tmux 快捷键

## 基础配置

| 配置项 | 值 |
|--------|-----|
| 前缀键 | `Ctrl` + `a` |
| 窗口起始编号 | 1 |
| 按键模式 | vi |
| 鼠标支持 | 开启 |
| 历史记录限制 | 50000 行 |

---

## 面板 (Pane)

| 快捷键 | 功能 |
|--------|------|
| `prefix` + `h` | 水平分割面板 |
| `prefix` + `v` | 垂直分割面板 |
| `prefix` + `x` | 关闭当前面板 |
| `Alt` + `←`/`→`/`↑`/`↓` | 切换面板 |
| `prefix` + `←`/`→`/`↑`/`↓` | 调整面板大小（可连续按） |

---

## 窗口 (Window)

| 快捷键 | 功能 |
|--------|------|
| `prefix` + `c` | 创建新窗口 |
| `prefix` + `r` | 重命名窗口 |
| `prefix` + `k` | 关闭当前窗口 |
| `Shift` + `←` / `→` | 切换窗口 |
| `Ctrl` + `Shift` + `←` / `→` | 交换窗口位置 |

---

## 会话 (Session)

| 快捷键 | 功能 |
|--------|------|
| `prefix` + `C` | 创建新会话 |
| `prefix` + `R` | 重命名会话 |
| `prefix` + `K` | 关闭当前会话 |
| `Shift` + `↑`/`↓` | 切换到上/下一个会话 |

---

## 复制模式

| 快捷键 | 功能 |
|--------|------|
| `prefix + [` | 进入复制模式 |
| `v` | 开始选择 |
| `y` | 复制并退出 |

---

## 其他

| 快捷键 | 功能 |
|--------|------|
| `prefix + p` | 重载配置文件 |
| `prefix + F` | fzf 会话切换 |

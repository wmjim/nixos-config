```fish
# 创建一个包含指定软件包的新 shell 环境
$ nix shell nixpkgs#cowsay nixpkgs#lolcat

# 现在可以使用软件包
$ cowsay hello, Nix! | lolcat
 _____________
< hello, Nix! >
 -------------
        \   ^__^
         \  (oo)\_______
            (__)\       )\/\
                ||----w |
                ||     ||
```

> [!TIP]
> 首次调用包，需要一些时间下载依赖项

**退出新 shell**：命令 `exit`、快捷键 `Ctrl` + `D`。

退出后指定软件包不再可用。

```nix
# 1. 单行字符串：双引号闭合
"Hello, nix!\n"

# 2. 多行字符串：两个单引号闭合
''
This is the first line.
This is the second line.
This is the third line.
''
```

Nix 的多行字符串会统一去除开头的缩进：

```nix
# 1. 智能缩进处理
''
  This is the first line.
  This is the second line.
    This is the third line.
''
# 处理后结果：每一行都被前移 最小缩进数 个字符
''
This is the first line.
This is the second line.
  This is the third line.
''
# 2. 处理占空
''

There's a row of spaces up there.
''
# 处理后结果
''
There's a row of spaces up there.
''
```

Nix 处理自动处理后的字符串，而不是原始字符串。

## 字符串插值

通过 `${}` 将名称的值插入到字符串中。

```nix
let
  name = "Nix";
in
"hello ${name}" # "hello Nix"
```

字符串插值语法支持的值必须为**字符串类型**。

## 字符串转义

```nix
# 单行字符串转义：使用 \
"this is a \"string\" \\"

# 多行字符串转义：使用 ''
let
  a = "1";
in
''the value of a is:
  ''${a}
  ''\n
''
# 会输出原始字符 ${a}，而不是做字符串插值
```

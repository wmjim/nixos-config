属性集是 `名称-值` 对的集合，其中名称在集合中唯一。

```nix
{
  string = "hello";
  integer = 1;
  float = 3.141;
  bool = true;
  null = null;
  list = [ 1 "two" false ];
  attribute-set = {
    a = "hello";
    b = 2;
    c = 2.718;
    d = false;
  };
}
```

```json
{
  "string": "hello",
  "integer": 1,
  "float": 3.141,
  "bool": true,
  "null": null,
  "list": [1, "two", false],
  "object": {
    "a": "hello",
    "b": 1,
    "c": 2.718,
    "d": false
  }
}
```


- **属性**：属性集内的名称
- **元素**：属性集内的 `名称-值` 对
- 属性集以 `{}` 为边界，结尾必须添加 `;`。

## 递归属性集

普通属性集不支持递归引用：

```nix
{ 
  a = 1;
  b = a + 1;
}
```

```bash
$ nix-instantiate --eval --strict file.nix
error: undefined variable 'a'
       at /home/mengw/examples/nix-examples/file.nix:3:7:
            2|   a = 1;
            3|   b = a + 1;
             |       ^
            4| }
```

即使 `a` 先定义，也无法访问。

```bash
# 递归属性集：前面添加 rec
rec {
  a = 1;
  b = a * 2 + 1;
}
```

```bash
$ nix-instantiate --eval --strict file.nix
{ a = 1; b = 3; }
```

## 属性访问

属性集中的属性可以利用 `.` 表示，这被称为**属性访问**。

```nix
let
  a = { b = { c = 123; }; };
in
  a.b.c

# 等价写法
let
  a.b.c = 123;
  # a = { b.c = 123; };
  # a.b = { c = 123; };
in
  a.b.c

```

1. 路径至少包含 1 个 `/`
2. 路径不能以 `/` 结尾

3. 绝对路径以 `/` 开头
4. 用 `..` 指定上级目录

```nix
# 根目录写法
/.
# /etc/file.txt
/etc/file.txt/.
/etc/file.txt
# 当前目录
./.
# 当前目录下的 download
./download
download/.
```

## 检索路径

通过系统变量来获取路径，使用 `<>` 包裹所需内容。

```nix
<nixpkgs>
```

```bash
/nix/var/nix/profiles/per-user/root/channels/nixpkgs
```

```nix
<nixpkgs/lib>
```

```bash
/nix/var/nix/profiles/per-user/root/channels/nixpkgs/lib
```

在生产代码中应[避免使用查找路径](https://nix.dev/guides/best-practices#search-path)，因为它们属于[不可复现的杂质](https://nix.dev/tutorials/nix-language#impurities)。

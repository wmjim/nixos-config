函数由参数和函数体组成，由 `:` 分隔。

```nix
# f(x) = x + 1
x: x + 1 # 匿名函数 <LAMBDA>
```

冒号左边的 `x` 是参数，右边的 `x+1` 是函数体。

```nix
# 将函数绑定到名称 f
let
  f = x: x + 1;
in
  f 2 # 2 作为函数调用参数，3
```

多元函数

```nix
# f(x, y) = 3x + y/2
# 函数在形式上只能有一个参数，但该参数可以是属性集
let
  f = { x, y }: ( 3 * x ) + ( y / 2 );
in
  f { x = 1; y = 4; } # 5

{ a = 1; b = 2; } # 标准属性集
```

在**函数定义**中作为参数出现的属性集，只包含属性名称，用 `,` 分隔。

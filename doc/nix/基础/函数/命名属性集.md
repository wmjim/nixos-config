若一个属性集没有与名称绑定，则称其为匿名属性集。

若一个属性集与名称绑定，则称其为命名属性集。

```nix
# {x, y} 匿名属性集
let
  f = { x, y }: ( 3 * x ) + ( y / 2 );
in
  f { x = 1; y = 4; }

# A 命名属性集
let
  # 用命名属性集 A 代替了匿名属性集 { x, y }
  # 同时 x、y 也要改用属性访问的写法 A.x、A.y
  f = A: ( 3 * A.x ) + ( A.y / 2 );
in
  f { x = 1; y = 4; }
```

函数的参数可以是**一个****命名属性集**与**一个****匿名属性集**的结合，以 `@` 连接。

此外**匿名属性集**必须包含 `...` 以允许额外属性。

```nix
let
  f = { x, ... }@A: ( 3 * x ) + ( A.y / 2 );
  # 也可以写成
  # f = A@{ x, ... }: ( 3 * x ) + ( A.y / 2 );
in
  f { x = 1; y = 4; }
```
